# パラメータ一覧

**実際のvalidationは `proto/` でprotovalidateを使って行っている**  

アプリケーションに登場する全てのパラメータの制約
先頭より `パラメータ名,受付型,下限,上限,備考` となっている。  
受付型は正規表現若しくは型で示される。下限・上限はそれぞれ示された値を含む境界値である。  

ユーザー名,[string],1,20,重複可能・再編集可能
ユーザーID,[UUIDv7],36,36,重複不可
ユーザーコード,[a-zA-Z0-9],4,20,重複不可・再編集可能
ルームID,[UUIDv7],36,36,重複不可
ルームコード,[0-9],6,6,重複不可・自動採番
