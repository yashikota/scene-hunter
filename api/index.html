Events

イベントストリームに接続する

イベントストリームに接続する

このエンドポイントは、Server-Sent Events (SSE) プロトコルを使用して、 リアルタイムのイベント通知をクライアントに送信します。

送信されるイベントタイプ:

  • room.player_joined: プレイヤーがルームに参加
  • room.player_left: プレイヤーがルームから退出
  • room.gamemaster_changed: ゲームマスターが変更
  • game.round_started: ラウンドが開始
  • game.hint_revealed: ヒントが公開
  • game.photo_submitted: 写真が提出
  • game.score_updated: スコアが更新
  • game.round_ended: ラウンドが終了
  • game.timer_update: タイマーが更新
Authorizations:
BearerAuth
path Parameters
room_id
required
string (Models.UUID)

UUID型

Responses

prd

https://scene-hunter.yashikota.com/api/events/{room_id}

http://localhost:4282/events/{room_id}
